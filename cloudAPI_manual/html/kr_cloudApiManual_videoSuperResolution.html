Video Super Resolution

저화질 영상의 해상도를 2배 높여 줍니다.

준비사항

- Input: 저화질 영상

  • 확장자: .mp4, .avi
  • 해상도: 720p 이하
  • 길이: 15초 이하
  • * fps가 높을수록 처리 속도가 느려집니다.
실행 가이드

① Request

  • Method : POST
  • URL : https://api.maum.ai/vsr/download

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
file Super Resolution 처리 할 영상 파일 file

③ Request 예제

curl --location --request POST 'http://api.maum.ai/vsr/download' \
--form 'apiId= {발급받은 API ID}' \
--form 'apiKey= {발급받은 API KEY}' \
--form 'file={동영상 파일 path}'

④ Response 예제

(.mp4 File Download)
