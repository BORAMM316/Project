차량 번호판 인식 (License Plate Recognition)

도로상에서 달리는 차량의 번호판 위치를 검출하고 번호를 인식해줍니다.

준비사항

- Input: CCTV에서 찍힌 차량 이미지 파일

  • 확장자: .jpg, .png.
  • 용량: 2MB 이하
실행 가이드

① Request

  • Method : POST
  • URL : http://api.maum.ai/smartXLoad/PlateRecog

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
car_img 차량 이미지 파일type:file (.jpg,.png)  string

③ Request 예제

curl --location --request POST 'http://dev-api.maum.ai/smartXLoad/PlateRecog' \
    --header 'Content-Type: multipart/form-data' \
    --form 'apiId= 발급받은 API ID' \
    --form 'apiKey= 발급받은 API KEY' \
    --form 'car_img= 번호판 이미지 파일'
