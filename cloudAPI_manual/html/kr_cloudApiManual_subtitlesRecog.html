이미지 자막 인식

이미지 중 자막을 인식하여 추출해줍니다. 현재는 한국어만 인식합니다.

준비사항

- Input: 이미지 파일

  • 확장자: .jpg, .png.
  • 용량: 5MB 이하
실행 가이드

① Request

  • Method : POST
  • URL : https://api.maum.ai/Vca/VCASubtitleExtract

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
scene_img type:file (.jpg, .png) 이미지 파일 string

③ Request 예제

curl --location --request POST 'http://api.maum.ai/Vca/VCASubtitleExtract' \
--header 'Content-Type: multipart/form-data' \
--form 'apiId= 발급받은 API ID' \
--form 'apiKey= 발급받은 API KEY' \
--form 'scene_img= 자막 추출할 이미지 파일'

④ Response 예제

{
    "result_img": "/9j/4AAQSkZJRgABAQAAAQABAAD/2wBDAAIBAQEBAQIBAQECAg...",
    "result_text": "별로 볼 거 없네"
}
