치아 교정기 포지셔닝 (Bracket Positioning)

주어진 치아에 교정기를 부착할 위치를 선정합니다.

준비사항

- Input: 치아 .obj 파일

  • 확장자: .obj
  • 치아 모델이 수직으로 정렬되어 있어야 합니다.
실행 가이드

① Request

  • Method : POST
  • URL : https://api.maum.ai/bracket/downloadZip

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
file 치아 obj 파일 file

③ Request 예제

curl --location --request POST 'http://api.maum.ai/bracket/downloadZip' \
--form 'apiId= {발급받은 API ID}' \
--form 'apiKey= {발급받은 API KEY}' \
--form 'file= {obj 파일 path}'

④ Response 예제 (Zip File Download)

확장자: .obj
ab: 브라켓  region: 브라켓 선정 위치
