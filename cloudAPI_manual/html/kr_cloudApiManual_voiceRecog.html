화자 인증(Voice Recognition)

화자의 목소리를 512차원 벡터화하여 그 값을 저장하고, 대조하여 인식해줍니다.

준비사항

Input: 특정 화자의 음성 파일

  • 확장자: .wav
  • Sample rate: 16000
  • Channels: mono
  • 용량: 2MB 이하
실행 가이드 ​SetVoice : 특정 화자의 목소리를 512차원 벡터화하여 지정한 DB에 저장합니다. (API 서버 내에 Json 타입으로 저장)

① Request

  • Method : PUT
  • URL : https://api.maum.ai/dap/app/setVoice

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
voiceId 해당 화자 음성에 부여할 Voice ID string
file type:file (.wav) 등록할 화자의 음성 파일 string
dbId 해당 Vector를 저장 할 DB의 ID (생략 시 ‘default’로 자동 저장) string

③ Request 예제

curl -X PUT \
http://api.maum.ai/dap/app/setVoice \
-H 'content-type: multipart/form-data; boundary=----
WebKitFormBoundary7MA4YWxkTrZu0gW' \
-F apiId=(*ID 요청 필요) \
-F apiKey=(*key 요청 필요) \
-F dbId=sample \
-F 'voiceId=샘플' \
-F 'file=@/sample.wav'

④ Response 예제

{
"message": {
"message": "success",
"status": 0
}
}

​recogVoice : 화자의 목소리를 512차원 벡터화하여 DB 내의 데이터와 대조합니다.

① Request

  • Method : POST
  • URL : https://api.maum.ai/dap/app/recogVoice

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
dbId 조회할 DB의 ID (생략 시, ‘default’로 자동 조회) string
file type:file (.wav) DB내 벡터값과와 대조할 화자의 음성 파일 string

③ Request 예제

curl -X POST \
http://api.maum.ai/dap/app/recogVoice \
-H 'content-type: multipart/form-data; boundary=----
WebKitFormBoundary7MA4YWxkTrZu0gW' \
-F apiId=(*ID 요청 필요) \
-F apiKey=(*key 요청 필요) \
-F dbId=sample \
-F 'file=@/sample2.wav'

④ Response 예제
