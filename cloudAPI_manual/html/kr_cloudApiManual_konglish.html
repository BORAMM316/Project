한글 변환 (Konglish)

영어 단어 또는 한글과 영어가 혼용된 문장에서 영어 단어들을 외래어 표기법에 가까운 한글로 변환시켜줍니다.

준비사항

① Input: 영어 단어 또는 영어와 한글이 혼용되어 있는 문장

실행 가이드 : 영어 단어

① Request

  • Method : POST
  • URL : https://api.maum.ai/api/konglish/words

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
wordList 콩글리시로 변환할 영어 단어 string

③ Request 예제

curl -X POST \
    http://api.maum.ai/konglish/words \
    -H 'Content-Type: application/json' \
    -H 'cache-control: no-cache' \
    -d '{
        "apiId": {발급받은 API ID},
        "apiKey":{발급받은 API KEY},
        "wordList": [
            "apple","bee"
        ]
    }

④ Response 파라미터 설명

Response
키 설명 타입
message API 동작여부 list
result 콩글리시로 변환된 영어 단어 목록 list
message: API 동작여부
키 설명 타입
message 요청 처리 상태를 설명하는 문자열 (Success/ Fail) string
status 요청 처리 상태에 대한 status code (0: Success) number

⑤ Response 예제

{
    "message": {
        "message": "Success",
        "status": 0
    },
    "result": [
        "애플",
        "비"
    ]
}
실행 가이드 : 영어 문장

① Request

  • Method : POST
  • URL : https://api.maum.ai/api/konglish/sentence

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
sentence 콩글리시로 변환할 영어 문장 string

③ Request 예제

curl -X POST \
    http://api.maum.ai/konglish/sentence \
    -H 'Content-Type: application/json' \
    -H 'cache-control: no-cache' \
    -d '{
        "apiId": {발급받은 API ID},
        "apiKey": {발급받은 API KEY},
        "sentence": "항상 ladyfirst이다"
    }'

④ Response 파라미터 설명

Response
키 설명 타입
message API 동작여부 list
result 콩글리시로 변환된 영어 문장 결과 string
message: API 동작여부
키 설명 타입
message 요청 처리 상태를 설명하는 문자열 (Success/ Fail) string
status 요청 처리 상태에 대한 status code (0: Success) number

⑤ Response 예제

{
    "message":{
        "message":"Success","status":0
    },
    "result": "항상 레이디퍼스트이다"
}
