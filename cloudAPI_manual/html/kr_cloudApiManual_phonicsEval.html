파닉스 평가

영어 단어의 특정 음소를 지정하여 음소 단위의 발음 평가를 제공합니다.

준비사항

① Input: 음성파일 (.wav/.pcm)

  • type:file (wav, pcm) 16K mono 음성파일

② Input: 확인할 음소

실행 가이드

① Request

  • Method : POST
  • URL : https://api.maum.ai/edueng/phonics

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
userId 사용자의 ID string
model 사용할 모델 (baseline) string
answerText 정답 문장 string
chySym 확인할 음소 string
file type:file (wav, pcm) 16K mono 음성 파일 file

③ Request 예제

curl -X POST \
https://api.maum.ai/edueng/phonics \
-H 'content-type: multipart/form-data; boundary=----WebKitFormBoundary7MA4YWxkTrZu0gW' \
-F apiId= (*ID 요청 필요) \
-F apiKey= (*key 요청 필요) \
-F userId= (사용자 ID) \
-F model= (사용할 model) \
-F answerText= (정답 text) \
-F chkSym= (확인할 음소) \
-F 'file=@eng_16k_1.wav' \

④ Response 파라미터 설명

Response
키 설명 타입
resultText 인식 문장 string
resCode API 응답 결과 (Success : 200) int
answerText 정답 문장 string
recordUrl 인식한 파일 url string
userPron 인식된 음소 string
correctPron 정답 음소 string
sentenceScore 정답 문장과 인식 문장 비교 점수 int
status 음소의 정답 여부 int
