Face-to-Face Avatar

동영상 내 인물의 얼굴 움직임을 포착하여, 사진 속 특정 인물이 이를 따라 움직이는 영상을 만드는 엔진입니다.

준비사항

- Input 1: 얼굴의 움직임이 있는 영상 파일

  • 확장자: .mp4
  • 용량: 30MB 이하
  • 길이: 20초 이하

- Input 2: 특정 인물의 얼굴 이미지 파일

  • 확장자: .jpg, .png
  • 용량: 2MB 이하
  • 얼굴이 차지하는 면적이 전체 이미지의 ¼ 이상
실행 가이드

① Request

  • Method : POST
  • URL : https://api.maum.ai/avatar/download

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
video 얼굴의 움직임 영상 파일 file
images 특정 인물의 얼굴 이미지 파일 (최대 5개) file

③ Request 예제

curl --location --request POST 'http://api.maum.ai/avatar/download' \
--form 'apiId= {발급받은 API ID}' \
--form 'apiKey= {발급받은 API KEY}' \
--form 'images= {이미지 파일 path}' \
--form 'images= {이미지 파일 path}' \
--form 'images= {이미지 파일 path}' \
--form 'images= {이미지 파일 path}' \
--form 'images= {이미지 파일 path}' \
--form 'video= {영상 파일 path}'

④ Response 예제

(.mp4 File Download)
