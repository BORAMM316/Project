이상행동 감지 (역주행 감지)

비디오 중 이상행동 (역주행)을 감지하여 알려줍니다.

준비사항

- Input: 비디오 파일

  • 확장자: .mp4
  • 용량: 50MB 이하
실행 가이드

① Request

  • Method : POST
  • URL : https://api.maum.ai/smartXLoad/anomalyDetect

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
video type:file (.mp4) 비디오 파일 string

③ Request 예제

curl --location --request POST 'http://api.maum.ai/smartXLoad/anomalyDetect' \
--header 'Content-Type: multipart/form-data' \
--form 'apiId= 발급받은 API ID' \
--form 'apiKey= 발급받은 API KEY' \
--form 'video= 역주행 비디오 파일'
