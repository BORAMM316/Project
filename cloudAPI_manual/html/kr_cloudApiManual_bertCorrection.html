문장 교정 (Bert Correction)

잘못된 한글 문장을 문맥에 맞게 교정해줍니다.

준비사항

① Input: 한글 문장

실행 가이드

① Request

  • Method : POST
  • URL : https://api.maum.ai/correct/sentence

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
sentence 문장 교정을 진행할 문장 (120자 이하) string

③ Request 예제

curl --location --request POST 'http://api.maum.ai/correct/sentence' \
    --header 'Content-Type: application/json' \
    --data-raw '{
        "apiId": "{발급받은 API ID}",
        "apiKey": "{발급받은 API KEY}",
        "sentence": "한꾹인뜰만 알아뽈 쑤 있께 짝썽하껬씁니따."
    }'

④ Response 파라미터 설명

Response
키 설명 타입
message API 동작여부 list
result 문장 교정 진행 결과 문장 string
message: API 동작여부
키 설명 타입
message 요청 처리 상태를 설명하는 문자열 (Success/ Fail) string
status 요청 처리 상태에 대한 status code (0: Success) number

⑤ Response 예제

{
"message": {
"message": "Success",
"status": 0
},
"result": "한국인들만 알아볼 수 있게 작성하겠습니다."
}
