침입 감지 (Intrusion Detection)

영역을 지정하여 해당 영역을 누군가가 침입할 시 알려줍니다.

샘플 파일로 테스트 하기

내 파일로 해보기

  • * 지원가능 파일 확장자: .mp4
  • * 동영상 파일 용량 50MB 이하만 가능합니다.
  • * CCTV 환경에 맞게 촬영된 영상이어야 합니다.
  • * 사람이 잘리지 않고 프레임 안에 위치하고 있어야 합니다.
  • * 샘플과 비슷한 각도 및 조도 영상을 사용해주세요.
  • * 영상 내 감지할 대상이 한 명일 경우, 정확도가 높습니다.

영역 설정

* 침입을 감지할 영역을 지정해주세요. (ex. 차선 도로, 개인 사유지 등)

침입 감지 (Intrusion Detection)

특정 지역에 침입하는 사람을 인식합니다.

준비사항

- Input: 비디오 파일

  • 확장자 : .mp4
  • 용량 : 50MB 이하
실행 가이드

① Request

  • Method : POST
  • URL : https://api.maum.ai/abnormal-behavior/intrusion:detect

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
video type:file (.mp4) 비디오 파일 file
roiList 관심 영역 정보, 침입을 감지할 영역 정보 string

③ Request 예제 

curl --location --request POST 'https://api.maum.ai/abnormal-behavior/intrusion:detect' \
--header 'Content-Type: multipart/form-data' \
--form 'apiId= 발급받은 API ID' \
--form 'apiKey= 발급받은 API KEY' \
--form 'video= 침입을 감지할 비디오' \
--form 'roiList="{\"overall\":[0,130,1280,130,1280,720,0,720]}"'

④ Response 예제 

{
	"message": {
		"message": "Success",
		"status": 0
	},
	"payload": {
		"status": "intrusion",
		"abnormalImg": "w7/DmMO/w6AAEEpGSUYAAQEAAAEAAQAAw7/..."
	}
}
