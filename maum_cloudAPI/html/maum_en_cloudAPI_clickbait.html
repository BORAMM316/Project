A씨는 체포된 이후에도 영어로만 말을 하는 등 이상 행동을 보인 것으로 알려졌다. 경찰 관계자는 “A씨가 방화 혐의를 전면 부인하고 있다며 “붙을 지른 동기와 방화 방법을 조사 중이다. 현장에서 유증반응이 없어 인화성 물질을 뿌린 것으로는 보이지 않는다＂고 설명했다.

Clickbait News

If the title of the news and the content of the body are different, it determines whether it is a clickbait news.

Preparation

- Input: Sentence (Korean)

API Document

① Request

  • Method : POST
  • URL : https://api.maum.ai/fake/prediction:make

② Request parameters

Name Description type
apiId Unique API ID. Request from is required for the ID. string
apiKey Unique API key. Request from is required for the key. string
title article title string
description article body string

③ Request example

curl --location --request POST 'http://api.maum.ai/fake/prediction:make' \
--header 'Content-Type: application/json' \
--data-raw '{
	"apiId": "{Own API Id}",
	"apiKey": "{Own API Key}",
	"title": " \"중국 바이러스\" LA 한인타운서 한국계 남 무차별 폭행 당해",
	"description": "미국 로스앤젤레스 한인타운에서 한국인으로 추정되는 아시 아계 남성이 인종차별적 발언과 함께 폭행을 당했다. 미 경찰은 신종 코로나바이러스 감염증(코로나19) 혐오범죄 가능성을 두고 수사에 나섰다. 최근 미국에선 코로나19 팬데믹(세계적 대유행) 기간 매일 한인을 대상으로 한 증오범죄 사건이 발생했다는 분석도 나왔다. 또한 허창수 회장은 여러 가지로 힘든 환경 속에서 전경련을 잘 이끌어 왔고, 앞으로도 국내외적으로 경제여건이 좋지 않은 상황에서 전경련과 민간 경제계를 이끌어 갈 수 있는 최적임자라는 것이 일치된 의견이었다고 덧붙였다."
}’

④ Response parameters

Response
Name Description type
message Status of API request string
payload Converted sentence array string
message: API operation
Name Description type
message String describing the request processing status (Success/Fail) string
status Status code for request processing status (0: Success) int
payload: result
Name Description type
category String that judges and explains clickbait (none/Clickbait) string
label status code for clickbait (0: none, 1: clickbait) int
reliability A number representing the probability of clickbait float

⑤ Request example

{
	"message": {
		"message": "Success",
		"status": 0
	},
	"payload": {
		"category": "clickbait",
		"label": "1",
		"reliability": 0.9981736
	}
}
