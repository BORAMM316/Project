Speech Recognition (Speech-To-Text)

Minds Lab’s STT API transforms speeches into texts

Preparation

① Input: Sound file (.wav/.pcm/.mp3)

② Model: Choose one below

  • Korean for customer service center (baseline, kor, 8000)
  • Korean for general use (baseline, kor, 16000)
  • English for customer service center (baseline, eng, 8000)
  • English for general use (baseline, eng, 16000)
  • Korean Address (address, kor, 8000)
API Document

① Request

  • Method : POST
  • URL : https://api.maum.ai/api/stt/

② Request parameters

Name Description type
ID Unique API ID. Request from is required for the ID. string
key Unique API key. Request from is required for the key. string
cmd Command for the API (runFileStt) string
lang Selected language ( kor / eng ) string
sampling Selected sampling rate (8000 / 16000) string
level Selected level (baseline / address) string
file Input sound file (wav / pcm / mp3) string

③ Request example 

		curl -X POST \ https://api.maum.ai/api/stt/ \
		  -H 'content-type: multipart/form-data;
		  boundary=----WebKitFormBoundary7MA4YWxkTrZu0gW' \ -F ID= (*Request for ID) \
		  -F key= (*Request for key) \
		  -F lang=eng \
		  -F sampling=8000 \ -F level=baseline \
		  -F 'file=@eng_8k_1.wav' \
		  -F cmd=runFileStt

④ Response parameters

Response
Name Description type
status Status of the API request (Success / Fail) string
extra_data Data of the input file object
data STT result (text) string
extra_data: Data of the input file
Name Description type
stt_length Character length of the text result int
stt_data STT result (text) string
stt_duration Duration of the input file (seconds) number

⑤ Response example 

		{
			"status": "Success",
			"extra_data": {
				"stt_length": 18,
				"stt_data": "Let's play badminton.\n",
				"stt_duration": "2.387"
			},
			"data": "Let's play badminton.\n"
		}
