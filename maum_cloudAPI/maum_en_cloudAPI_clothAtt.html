Clothing Multi-Attributes Detection

Recognizes the characteristics of the person's clothing in the image and displays the results.
Currently, the recognition rate is high only for the elderly.

Use My File

  • * Image capacity : 10MB or less
  • * Image size : 240x240 px or higher
  • * Must be at least one person in the image.
  • * Upper/lower/whole body are recognizable.

의상 특징 인식 (Clothing Multi-Attributes Detection)

이미지 내 인물의 의상 특징을 인식하고, 결과를 나타내줍니다.

준비사항

- Input: 이미지 파일

  • 확장자 : .jpg
  • 용량: 10MB 이하
  • 이미지 사이즈 : 240x240px 이상
  • 이미지 속 사람이 1명 이상 있어야합니다.
  • 상반신/하반신/전신 모두 인식 가능합니다.
실행 가이드

① Request

  • Method : POST
  • URL : https://api.maum.ai/feat/getClothFeature

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
scene_img type : file(.jpg) 이미지 파일 string

③ Request 예제 

curl --location --request POST
'https://api.maum.ai/feat/getClothFeature' \
--header 'Content-Type: multipart/form-data' \
--form 'apiId= 발급받은 API ID' \
--form 'apiKey= 발급받은 API KEY' \
--form 'scene_img= 의상을 인식할 이미지 파일'
