키 발급

1) Mindslab에서 제공되는 API 서비스를 사용하기 위해서는 키를 선지급 받아야합니다.

2) 마음AI 플랫폼에서 Business Plan 이상 구독 신청을 하셔야합니다. (https://maum.ai)

3) [계정메뉴]에서 본인의 API ID 및 Key를 확인해주세요.

4) 발급받은 API ID 및 Key를 기억하여 아래 매뉴얼에 맞게 사용합니다.

※ Mindslab API를 위한 ID와 키는 외부 유출 및 공유를 금합니다.

TTS (Text-To-Speech)

마인즈랩의 TTS API는 텍스트 문장 또는 파일을 음성으로 변환시켜주는 API입니다.
20분 학습만으로도 다양한 모델로 자연스럽게 음성을 합성해주는 마인즈랩의 API를 경험해보세요.

준비사항

① Input: 텍스트(문장) *제약사항: 1000자 이내의 문장(한글/영어 기준)

② 아래 Model 중 택 1

  • 한국어 성인 여자 - 차분한 (kor_female1)
  • 한국어 성인 여자 - 발랄한 (kor_female2)
  • 한국어 성인 여자 - 자연스러운 (kor_female3)
  • 한국어 성인 여자 - 정직한 (kor_female4)
  • 한국어 성인 여자 - 개성있는 (kor_female5)
  • 한국어 성인 여자 - 친근한 (baseline_kor)
  • 한국어 성인 남자 - 자연스러운 (kor_male1)
  • 한국어 성인 남자 - 진중한 (kor_male2)
  • 한국어 성인 남자 - 차분한 (kor_male3)
  • 한국어 성인 남자 - 친근한 (kor_male4)

실행 가이드

① Request

  • Method : POST
  • URL : https://api.maum.ai/tts/stream

② Request 파라미터 설명

키 설명 type
apiId 사용자의 고유 ID. 마인즈랩 담당자에게 이메일로 발급요청 string
apiKey 사용자의 고유 key. 마인즈랩 담당자에게 이메일로 발급요청 string
text 문장 (*1000자 이내) string
voiceName 사용할 Voice Model Name
* 그 외 다른 모델 사용이 필요할 시에는 문의하세요 		string

③ Request 예제 

curl -X POST \
  https://api.maum.ai/tts/stream \
  -H 'Content-Type: application/json' \
  -d '{
    "apiId" : (*ID 요청필요),
    "apiKey" : (*Key 요청필요),
    "text" : 안녕하세요, 반갑습니다. Minds Lab 상담원입니다.,
    "voiceName" : baseline_kor
}'

④ Response 예제

audio/x-wav 타입의 chunked data (StreamBody)

적용사례 (Use Cases)

마인즈랩의 앞선 기술을 다양하게 활용할 수 있습니다.
  • CASE 01 AI 자동 해피콜
    설문조사, 상품 품질 조사와 같은 반복적인 성격의 전화를 인공지능으로 대신하는 AI 자동콜 해피콜에 응용할 수 있습니다.
    • TTS
  • CASE 02 안내 서비스
    다양한 목적의 안내나 어시스턴트 서비스에 실제처럼 자연스러운 TTS 음성을 활용할 수 있습니다.
    • TTS
  • CASE 03 유튜버
    유튜브나 브이로그 등 영상 콘텐츠에 즉각적으로 반영할 수 있습니다.
    • TTS
  • CASE 04 게임
    게임 상의 등장인물 음성이나 음향 효과 등을 만들 수 있습니다.
    • TTS
  • CASE 05 아동 교육 콘텐츠
    아이들을 대상으로 한 교육용 콘텐츠에 접목할 수 있습니다.
    • TTS
  • CASE 06 뉴스 브리핑
    고객이 원하는 목소리로 개인화된 뉴스 브리핑 서비스를 제공할 수 있습니다.
    • TTS
    • XDC
