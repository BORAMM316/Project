API Guideline

Set up Environment

1) REST API is available wherever you can send HTTP requests.

2) It can be used in Python, Web, Javascript and Java.

ID & Key

1) You need a unique ID and key for Minds Lab’s API services.

2) Subscribe ‘Business Plan’ or ‘Enterprise Plan’ on maum.ai platform. (https://maum.ai)

3) After subscription, request an ID and key with required information filled in.
(ex. Service description, Requesting API, Company info. etc.)

4) After an agreement, Minds Lab would send you an ID, key within 1-2 business days through email.

5) Remember the ID and key and follow the instruction below.

Lip Sync Avatar

A naturally speaking avatar is created following the inputted text.

Preparation

- Input 1: 듣고 싶은 문장 텍스트

  • 제약사항 1) : 1000자 이내의 문장(한글 기준)
  • 제약사항 2) : 긴 문장을 넣을수록 동영상 처리하는 시간이 오래 걸립니다

- Input 2: 배경 이미지 파일

  • Supported files : .jpg, .png
  • Image capacity : 10MB or less
API Document ​Upload : 발화할 텍스트 및 배경 이미지를 넣고 request key를 받아옵니다.

① Request

  • Method : POST
  • URL : https://api.maum.ai/lipsync/upload

② Request parameters

Name Description type
apiId Unique API ID. Request from is required for the ID. string
apiKey Unique API key. Request from is required for the key. string
text 문장 (*50자 이내) file
image 배경 이미지 file

③ Request example 

curl --location --request POST
'https://api.maum.ai/feat/getHairInfo' \
--header 'Content-Type: multipart/form-data' \
--form 'apiId= Own API Id' \
--form 'apiKey= Own API Key' \
--form 'text= 아바타가 발화할 문장'
--form 'image= 아바타의 배경 이미지 파일'

④ Request parameters

Response
Name Description type
message API 동작여부 list
payload 결과 list
message: API 동작여부
Name Description type
message 요청 처리 상태를 설명하는 문자열 (Success/ Fail) string
status 요청 처리 상태에 대한 status code (0: Success) number
payload: 결과
Name Description type
requestKey 아바타 생성을 위한 request Key string

⑤ Response example 

{
"message": {
"message": "Success",
"status": 0
},
"payload": {
"requestKey": "요청 key"
}
}
